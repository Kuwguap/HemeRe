# 🗺 Project Roadmap

## Phase 1: Core MVP ✅
- [x] Basic task creation via `/add` command.
- [x] Persistent SQLite database integration.
- [x] Inline keyboard for "Mark Done" and "Delete".
- [x] Basic "Add Reminder" logic (15min, 1h, 3h, 1d via inline buttons).

## Phase 2: User Experience Enhancements
- [ ] **NLP Parsing:** Use Natural Language Processing to understand "Remind me to call Mom tomorrow at 5pm".
- [ ] **Task Categories/Tags:** Group tasks by #Work, #Personal, or #Urgent.
- [ ] **Edit Mode:** Update task descriptions or titles via inline buttons.

## Phase 3: Advanced Features (The Upgrades)
- [ ] **Voice-to-Task:** Send a voice note; the bot transcribes it into a task.
- [ ] **Sub-tasks:** Add the ability to break big tasks into a checklist.
- [ ] **Priority Levels:** Color-coded emojis based on task urgency (🔴, 🟡, 🟢).
- [ ] **Recurring Tasks:** Support for daily/weekly habits.

## Phase 4: Data & Integration
- [ ] **Export/Backup:** Export tasks to CSV or JSON.
- [ ] **Web Dashboard:** A simple site to view tasks outside of Telegram.
- [x] **Multi-user Support:** Tasks are scoped per user_id.
